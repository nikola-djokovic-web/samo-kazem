# **App Name**: SamoKazem Central

## Core Features:

- Admin Dashboard: Full CRUD (Create, Read, Update, Delete) interface for administrators to manage comedy blog posts.
- Landing Page: Displays the newest and featured comedy blog posts for guest users.
- All Posts Page: Displays all available comedy blog posts in reverse chronological order.
- Single Post Page: Displays the full content of a single comedy blog post with options to like and comment.
- Favorite Posts: Allows users to save their favorite comedy posts.
- User Authentication: Allows registered users to create an account and manage saved posts and comments.
- User comments: Allow the user to save the comments about the blog post in the database

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) for a modern and engaging feel.
- Background color: Light gray (#F0F2F5) for a clean, minimalistic backdrop.
- Accent color: Electric purple (#8E2DE2) to highlight key actions and interactive elements.
- Body text: 'PT Sans' for clear and accessible readability.
- Headline font: 'Playfair' for headings, for an elegant and readable feel. Note: if longer text is anticipated, use 'PT Sans' for body.
- Use a consistent set of minimalistic icons for navigation and actions.
- Grid-based layout to provide clear sections, modern feel, and mobile responsiveness.
- Subtle thumbnail animations (zoom/slide), and smooth button hover effects.
